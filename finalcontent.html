AJAX Technology and its Applications

Toda’s project is devoted to a mechanism for Web Site Page Managing called AJAX. AJAX is not really a technology in itself, but rather an approach to using a number of existing technologies together, including HTML or XHTML, CSS, JavaScript, DOM, XML, XSLT, in order to make quick, incremental updates to the user interface without reloading the entire browser page. The task will consist on first – understand how the AJAX is implemented, second – train AJAX on some examples, and third – implement the AJAX mechanism in a new Website in order to manage the webpages which have been designed so far or transform them so that some AJAX mechanism will be present within.
